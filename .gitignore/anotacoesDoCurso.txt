--> Estas sao anotacoes do curso de js aula 4:

----------------
== significa comparacao implicita ou também comparacao de valor

  como foi dito ele compara o valor entao se a string "5" é o mesmo valor do number (5) entao da true, mas se em algum dos 2 estiver um valor 
diferente entao da false por exemplo uma string "6"

=== significa comparacao explicita, no caso ele pega o valor das variaveis e compara o tipo e valor 

atualmente é boa prática usar os 3 iguais(===) e quando for realizada conversoes faze-la de modo explicita

operador ternario, é chamado assim por ter tres operadores numa linha de codigo

  nao é recomendado coloca varios operadores ternarios dentro do outro pois fica dificil a leitura e compreensao, nesse caso usar o if. é uma boa 
prática

  Uma função é criada para ser executado quando eu quizer podendo ser utilizada uma ou mais vezes no decorrer do codigo. Primeiro passa o declarar
a função em seguida chamar a função para ser executada. Há 3 formas de declarar função. É possivel chamar umas funcao dentro de outra, basta ao 
chamar a funcao coloca dentro dos parentesis a outra funcao.
